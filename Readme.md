# is

  `is` is a type checking component

[![Build Status](https://travis-ci.org/codeactual/is.png)](https://travis-ci.org/codeactual/is)

## Installation

```
$ component install codeactual/is
```

## API

- is.array
- is.arguments
- is.Function
- is.string
- is.number
- is.date
- is.regexp
- is.object

Based on the underscore.js type checking functions.
